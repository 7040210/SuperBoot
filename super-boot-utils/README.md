# 工具模块
　　工具模块提供一些基础功能，因为目前发现第三方开源项目hu-tool做的工具模块很棒，里边集成了大量的功能方法，基于不重复造轮子的原则，后期部分业务使用上的场景使用了hu-tool的工具模块的功能。


### 引入方式

```xml

    <!-- 工具模块 -->
    <dependency>
        <groupId>org.superboot</groupId>
        <artifactId>utils</artifactId>
        <version>1.0-SNAPSHOT</version>
    </dependency>
        
```
